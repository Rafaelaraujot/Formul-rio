Perguntas e Respostas

  1. Na tag input com atributo type="text", para que serve o atributo "placeholder"?

    O atributo "placeholder" serve para exibir um texto dentro do campo de input antes do usuário digitar algo. Esse texto geralmente orienta o usuário sobre o que deve ser preenchido.

  2. Na tag input com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

    O atributo "checked" define que o item estará previamente marcado ao carregar a página.

  3. Na tag input para que serve o atributo "value"?

    O atributo "value" define o valor padrão do campo de entrada, ou seja, o valor que será enviado ao servidor quando o formulário for submetido.

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    O atributo "name" deve ser o mesmo para agrupar os botões de rádio, permitindo que apenas um deles seja selecionado por vez.

  5. Para que serve a tag label? Para que serve o atributo "for" nessa tag?

    A tag label é usada para associar um texto a um campo de formulário. O atributo "for" conecta o label ao id de um campo específico, facilitando a acessibilidade.

  6. Na tag option para que serve o atributo "selected"?

    O atributo "selected" define qual opção estará selecionada por padrão em um elemento select.

  7. Nas tags de formulário para que serve o atributo "required"?

    O atributo "required" torna o preenchimento de um campo obrigatório antes do envio do formulário.

  8. Para que serve a tag textarea?

    A tag textarea é usada para criar uma área de texto maior onde os usuários podem digitar textos longos.

  9. Na tag form para que serve o atributo "action"?

    O atributo "action" define para onde os dados do formulário serão enviados quando ele for submetido.

  10. Na tag form para que serve o atributo "method"?

    O atributo "method" define o método HTTP usado para enviar os dados do formulário (GET ou POST).

  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?

    Deve ser utilizado o método "POST".

  12. Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?

    O valor deve ser "POST".

  13. O que as tags fieldset e legend produzem?

    As tags fieldset criam um agrupamento visual de campos em um formulário, enquanto a tag legend fornece um título para esse grupo.
